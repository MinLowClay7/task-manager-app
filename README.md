# Task Manager Desktop

Aplicación de escritorio desarrollada en Python con PySide6.

## Features
- CRUD de tareas
- Persistencia local en JSON
- Dark / Light mode
- Arquitectura MVC
- UI moderna

## Tecnologías
- Python
- PySide6 (Qt)
- MVC
- Git

## Screenshots


## Cómo ejecutar
pip install -r requirements.txt
python main.py

## Distribución
La aplicación se distribuye como ejecutable para Windows generado con PyInstaller.

> El ejecutable no se incluye en el repositorio.
